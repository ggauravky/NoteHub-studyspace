Unit 1: Networking

Internet, World Wide Web, Web servers, Web clients, Web sites and Web pages, Web browsers, blogs, news groups, HTML, web address, e-mail address, downloading and uploading files from a remote site. Introduction to web page designing using HTML, creating and saving an HTML document, accessing a web page using a web browser.
